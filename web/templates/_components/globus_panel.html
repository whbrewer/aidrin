Endpoint Setup Guide

On the remote machine where your data is located:

# Install
pip install globus-compute-endpoint aidrin
# Configure & start
globus-compute-endpoint configure aidrin-ep
globus-compute-endpoint start aidrin-ep
# Get endpoint UUID
globus-compute-endpoint list
# Stop when done
globus-compute-endpoint stop aidrin-ep

Copy the Endpoint ID from the list output and paste it above.
